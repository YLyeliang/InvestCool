# InvestCool 项目变更与规范文档

## 1. 文章类目标准化 (2026-03-19)
为了保持投研内容的专业性与一致性，`/analysis` 页面（投资分析板块）仅允许使用以下四大标准类目。严禁私自增加或修改类目名称。

### 标准类目列表：
- **投资入门**：基础知识、开户指南、QDII 购买工具等。
- **深度分析**：行业综述、长篇趋势研报、技术面分析等。
- **公司基本面**：个股财报拆解、科技巨头业务调研等。
- **AI**：AI 辅助生成的实时简评、大模型策略分析等。

### 约束规则：
1. 任何新生成的分析文章必须归属于上述四类之一。
2. InvestCool 系统本身的「技术教程」、「架构说明」必须归类到 `tutorials`（教程）板块，不得占用分析类目。

## 2. UI 规范 (Spike Pro 风格)
- 全局圆角：`var(--radius-lg)` (16px)。
- 阴影标准：优先使用 `var(--shadow-sm)` 和 `var(--shadow-lg)`。
- 字体：`Plus Jakarta Sans` / `Inter`。
